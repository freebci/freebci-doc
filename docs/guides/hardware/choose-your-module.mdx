# Choose Your Module

> Start with the module that matches your goal, then move up to the full multichannel workflow on the same DAQ platform.

| Module | Best For | What You Get | Typical Outcome |
|---|---|---|---|
| **FreeBCI Spike 1CH LH001** | Beginners, makers, classroom demos, fast iteration | Single-channel biopotential acquisition | First signal, first waveform, focus / engagement demo |
| **FreeBCI Spike 8CH ADS1299** | EEG builders, researchers, multichannel prototyping | Eight-channel EEG acquisition | Multichannel waveforms, site-aware setup, richer analysis |

## Start with 1CH if...

- You want the fastest path to a working FreeBCI system
- You want to demo **Engagement Index** and binary **focus** classification
- You are learning electrode basics, signal quality, and DAQ workflow
- You want a lower-complexity bring-up path before moving to multichannel EEG

Go to: [Spike 1CH LH001](/docs/hardware/spike-1ch-lh001)

## Choose 8CH if...

- You already understand the basics and want full EEG channel mapping
- You need multiple electrode sites connected at the same time
- You want a more complete spatial setup for monitoring and analysis
- Your goal is system prototyping, experimentation, or multichannel demos

Go to: [Spike 8CH ADS1299](/docs/hardware/spike-8ch-ads1299)

## Shared Software Platform

Both modules are supported by **FreeBCI DAQ**.

- Same browser-based acquisition workflow
- Same Web Serial connection model
- Same live monitoring, EI, focus, and AI analysis surfaces
- Different hardware complexity and setup depth

Start with the common DAQ workflow here: [Connect to FreeBCI DAQ](/docs/hardware/connect-to-freebci-daq)

## Recommended Learning Path

1. Read [Safety and Lab Setup](/docs/hardware/safety-and-lab-setup)
2. Start with [Spike 1CH LH001](/docs/hardware/spike-1ch-lh001) if you are new to EEG hardware
3. Run the [1CH Focus Demo](/docs/hardware/spike-1ch-lh001/focus-demo)
4. Move to [Spike 8CH ADS1299](/docs/hardware/spike-8ch-ads1299) when you need simultaneous multichannel acquisition
5. Follow [Upgrade from 1CH to 8CH](/docs/hardware/upgrade-from-1ch-to-8ch) when you are ready
