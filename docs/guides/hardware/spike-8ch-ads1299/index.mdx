# FreeBCI Spike 8CH ADS1299

> The multichannel path in FreeBCI hardware: designed for fuller EEG setups, richer site mapping, and more complete analysis in FreeBCI DAQ.

## Positioning

FreeBCI Spike 8CH ADS1299 is the **advanced multichannel module** in the FreeBCI lineup.

It is best for:

- multichannel EEG prototyping
- channel-to-site mapping workflows
- richer live monitoring and analysis
- users graduating from the 1CH demo path

## Open Release Contents

The public release package includes:

- schematic PDF
- Interactive BOM HTML
- Gerber ZIP

It does **not** include editable EDA sources, firmware source, enclosure files, production test procedures, or regulatory documentation.

## Hardware Scope

This module is based on **ADS1299** and is intended for eight-channel acquisition workflows. Final electrical interpretation, connector definitions, and net-level details should always be verified against the released schematic.

## What Success Looks Like

A successful first-run path for this module is:

1. Board powers on cleanly
2. FreeBCI DAQ establishes a stable serial connection
3. Channel-to-site bindings are configured correctly
4. Multiple live waveforms stream together
5. Monitoring and analysis reflect a true multichannel setup
