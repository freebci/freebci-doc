# FreeBCI Hardware

> Start with a single-channel focus demo, then grow into full multichannel EEG on the same DAQ platform.

FreeBCI hardware currently centers on two paths:

- **FreeBCI Spike 1CH LH001** — the onboarding path for makers, beginners, and fast focus / engagement demos
- **FreeBCI Spike 8CH ADS1299** — the multichannel path for fuller EEG acquisition and analysis

Both are supported by **FreeBCI DAQ**.

## Product Story

The intended progression is:

1. Start with **1CH** to learn the workflow and run the focus demo
2. Move to **8CH** when you need simultaneous multichannel acquisition
3. Keep the same **FreeBCI DAQ** workflow across both

Continue here: [Upgrade from 1CH to 8CH](/docs/hardware/upgrade-from-1ch-to-8ch)
