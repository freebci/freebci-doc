# Getting Started

New to FreeBCI? Start with [FreeBCI DAQ Quick Start](/docs/freebci-daq/quick-start) — the fastest way to connect your hardware and see live brainwaves in your browser.

## What You Need

- **Chrome or Edge** (desktop) — Web Serial API support
- **FreeBCI-compatible EEG hardware** — with USB serial output
- **FreeBCI DAQ** — [open-source software](https://github.com/freebci/freebci-daq) running in your browser

## Next Steps

1. [Quick Start](/docs/freebci-daq/quick-start) — 5-minute setup
2. [Hardware Setup](/docs/freebci-daq/hardware-setup) — Configure baud rate, gain, channels
3. [Live Monitoring](/docs/freebci-daq/live-monitoring) — View real-time EEG waveforms
