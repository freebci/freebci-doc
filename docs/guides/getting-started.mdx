# Getting Started

New to FreeBCI? Start by choosing the path that matches your goal.

## Path 1: 1CH Onboarding

Choose this if you want the fastest path to a working demo.

- Best for beginners, makers, and classroom demos
- Goal: power on safely, see a waveform, and run the focus / engagement demo

Start here:

1. [Start Here](/docs/start-here)
2. [Spike 1CH LH001](/docs/hardware/spike-1ch-lh001)
3. [1CH Focus Demo](/docs/hardware/spike-1ch-lh001/focus-demo)

## Path 2: 8CH Multichannel EEG

Choose this if you want a fuller EEG setup with multiple active channels.

- Best for advanced users, EEG builders, and multichannel prototyping
- Goal: bind multiple sites, validate multichannel streaming, and use richer monitoring

Start here:

1. [Start Here](/docs/start-here)
2. [Spike 8CH ADS1299](/docs/hardware/spike-8ch-ads1299)
3. [8CH Multichannel EEG Setup](/docs/hardware/spike-8ch-ads1299/multichannel-eeg-setup)
