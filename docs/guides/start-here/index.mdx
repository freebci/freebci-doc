# Start Here

> Choose the path that matches your goal: a 1-channel first-signal demo, a full 8-channel EEG setup, or the shared DAQ workflow that connects them.

## Choose Your Path

### 1CH: First Signal and Focus Demo

Best for:

- beginners
- makers
- classroom demos
- fast onboarding

Goal:

- power on safely
- connect one effective channel
- see a live waveform
- run the focus / engagement demo

Start with:

1. [Choose Your Module](/docs/hardware/choose-your-module)
2. [1CH Quick Start](#1ch-quick-start)
3. [Safety and Lab Setup](/docs/hardware/safety-and-lab-setup)
4. [Spike 1CH LH001](/docs/hardware/spike-1ch-lh001)
5. [Connect to FreeBCI DAQ](/docs/hardware/connect-to-freebci-daq)
6. [1CH Focus Demo](/docs/hardware/spike-1ch-lh001/focus-demo)

### 8CH: Full Multichannel EEG

Best for:

- advanced users
- EEG builders
- multichannel prototyping
- richer site-aware monitoring

Goal:

- set up the 8CH module safely
- bind multiple sites correctly
- validate multichannel streaming
- use FreeBCI DAQ for monitoring and tuning

Start with:

1. [Choose Your Module](/docs/hardware/choose-your-module)
2. [8CH Quick Start](#8ch-quick-start)
3. [Safety and Lab Setup](/docs/hardware/safety-and-lab-setup)
4. [Spike 8CH ADS1299](/docs/hardware/spike-8ch-ads1299)
5. [Connect to FreeBCI DAQ](/docs/hardware/connect-to-freebci-daq)
6. [8CH Multichannel EEG Setup](/docs/hardware/spike-8ch-ads1299/multichannel-eeg-setup)

## 1CH Quick Start

If your main goal is to get to a working demo fast:

1. Read [Safety and Lab Setup](/docs/hardware/safety-and-lab-setup)
2. Open [Spike 1CH LH001](/docs/hardware/spike-1ch-lh001)
3. Follow [1CH First Power-On](/docs/hardware/spike-1ch-lh001/first-power-on)
4. Follow [Connect to FreeBCI DAQ](/docs/hardware/connect-to-freebci-daq)
5. Run the [1CH Focus Demo](/docs/hardware/spike-1ch-lh001/focus-demo)
