# FreeBCI Documentation

Welcome to the FreeBCI documentation. The platform currently centers on two product lines:

- **FreeBCI Spike** — the hardware series
- **FreeBCI DAQ** — the browser-based acquisition and analysis software

If you are new here, start with the path that matches your goal.

- [Start Here](/docs/start-here) — Choose between the 1CH onboarding path, the 8CH multichannel path, and the shared DAQ workflow
