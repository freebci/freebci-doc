# FreeBCI DAQ

**浏览器端 EEG 脑电采集与实时频谱分析平台**

通过 USB 连接 EEG 硬件，在浏览器中实时查看 250 Hz 脑电波形，获取参与度与专注度指标。

## 系统要求

- **Chrome 或 Edge**（桌面版） — 需要 Web Serial API
- **localhost 或 HTTPS** — 需要安全上下文
- **EEG 硬件** 通过 USB 串口输出，波特率 921600

所有数据处理均在本地浏览器中完成，不会上传任何数据。
