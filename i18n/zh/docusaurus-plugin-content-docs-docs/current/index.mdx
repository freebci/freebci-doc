# FreeBCI 文档

欢迎来到 FreeBCI 文档。当前平台主要由两条产品线组成：

- **FreeBCI Spike** — 硬件系列
- **FreeBCI DAQ** — 浏览器端采集与分析软件

如果你是第一次来到这里，建议先从最适合你的路径开始。
