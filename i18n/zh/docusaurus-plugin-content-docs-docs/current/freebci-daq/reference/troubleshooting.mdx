# 故障排除

常见问题及解决方法。

## 浏览器支持

| 浏览器 | 支持？ |
|---|---|
| Chrome 89+（桌面版） | 是 |
| Edge 89+（桌面版） | 是 |
| Firefox、Safari、移动端 | 否 |

**"当前浏览器不支持 Web Serial"** → 切换至 Chrome 或 Edge。

## 连接问题

### 设备未找到

- 检查 USB 线缆和设备电源
- 检查操作系统串口驱动
- 尝试更换 USB 接口
- macOS 终端执行 `ls /dev/tty.*` 或 Windows 查看设备管理器

### ACK 超时错误

**"等待串口初始化 reset ACK 超时"** — 设备在 2 秒内未响应 EEGRST。检查固件兼容性、波特率（必须 921600）、是否有其他程序占用串口。

**"等待硬件初始化参数 ACK 超时"** — 设备拒绝了配置参数。尝试使用默认设置。

### 停滞检测

**"2 秒内未收到有效帧"** — 检查设备是否正在采样。可自动恢复。

## 数据流问题

启动流的四个前置条件：
1. 硬件参数已确认（绿色徽章）
2. 点位绑定已确认
3. 串口已连接
4. 如开启 CSV，需已选择输出文件

## AI 连接问题

- **Base URL 错误** → 检查是否与供应商匹配
- **API Key 无效** → 检查是否有多余空格
- **Ollama 未运行** → 执行 `ollama serve`
- **CORS 错误** → 端点必须支持跨域访问

## 性能问题

- 关闭 **Draw** 开关可暂停图表渲染，保持数据流动
- IndexedDB 写入超时 → 导出并清理旧对话

## 仍无法解决？

- 查看[系统与诊断](/zh/docs/freebci-daq/system-tuning)的运行环境状态
- 检查[硬件配置](/zh/docs/freebci-daq/hardware-setup)的前置条件
- 参阅[完整配置参考](/zh/docs/freebci-daq/reference/configuration)
- 查阅[常见问题](/zh/docs/freebci-daq/reference/faq)
