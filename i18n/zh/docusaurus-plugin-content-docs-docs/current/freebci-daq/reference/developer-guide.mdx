# 开发者指南

> 扩展 FreeBCI DAQ 的开发者入口说明。

本页作为中文开发者入口，帮助你理解结构与路径；更深入的实现教程目前仍以英文版为主。

## 开发者路径

如果你要修改或扩展 FreeBCI DAQ，建议按下面顺序进入：

1. [开发者教程总览](/zh/docs/freebci-daq/dev)
2. [数据流水线](/zh/docs/freebci-daq/reference/data-pipeline)
3. [配置参考](/zh/docs/freebci-daq/reference/configuration)

## 主要代码区域

| 路径 | 作用 |
|---|---|
| `src/transport/` | 串口与数据传输 |
| `src/algorithms/` | 参与度指数与频带分析 |
| `src/focus/` | 专注度状态机 |
| `src/components/` | UI 组件 |
| `src/store/` | 状态管理 |
