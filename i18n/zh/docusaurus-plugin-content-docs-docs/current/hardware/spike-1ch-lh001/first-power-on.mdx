# 1CH 首次上电

> 分阶段 bring-up，不要把第一次测试直接做成人体连接实验。

## 上电前

请检查：

- 板卡与目标模块版本一致
- 没有可见焊桥或反装器件
- 连接器方向正确
- 电池接线和极性正确
- 测试环境与不安全的市电实验设备隔离

## 首次 bring-up 目标

第一次 bring-up 的目标很简单：

- 板卡上电正常
- 没有异常发热
- 稳定到足以连接 FreeBCI DAQ

此阶段你验证的是**板卡本身**，而不是完整的人体信号链路。

## 推荐 bring-up 顺序

1. 不接电极的情况下给板卡上电
2. 观察是否有明显故障、不稳定或发热
3. 仅在隔离的 DAQ 工作流需要时接入 USB
4. 打开 FreeBCI DAQ 并尝试建立串口连接
5. 确认软件能够稳定识别设备

## 什么时候应该停止

如果你看到以下现象，请停止并检查：

- 频繁断连或状态不稳定
- 有发热迹象
- 可见装配缺陷
- 行为与预期模块发布不一致

在这些问题解决前，不要继续进入人体连接实验。
