# 1CH 专注度 Demo

> 这是 FreeBCI Spike 1CH LH001 的标志性入门流程：接入一个通道，流式连接到 FreeBCI DAQ，并跑通参与度指数 / 专注度 Demo。

## 你在演示什么

这个流程用来证明：

- 板卡能够稳定流式连接到 FreeBCI DAQ
- 信号链足够稳定，可以进行实时分析
- FreeBCI DAQ 能计算 **参与度指数（EI）**
- 内置 **专注度校准** 工作流可以完整跑通

## 搭建预期

对于 1CH Demo：

- 一个有效采集通道就足够
- 一个点位绑定就足够完成核心流程
- 重点是信号稳定性和 EI / 专注度表现，而不是空间覆盖

## DAQ 流程

1. 打开 FreeBCI DAQ
2. 在 **Setup（设置）** 中确认硬件参数
3. 添加一个采集点位绑定
4. 打开有线连接
5. 开始采集
6. 前往 **Live（实时监测）**
7. 再前往 **参与度与专注度**

如果你想先看通用软件流程，请先阅读 [连接到 FreeBCI DAQ](/zh/docs/hardware/connect-to-freebci-daq)。

## 正常情况下你会看到什么

一个健康的 Demo 运行应当表现为：

- **Live** 页面出现实时波形
- 滤波信号相较原始波形更干净
- 初始分析窗口填满后，FFT 相关图表开始更新
- 在预热和滚动窗口累积之后，**EI** 趋势开始变化

## 专注度校准

1CH 模块非常适合用来跑内置专注度 Demo，因为整体流程简单：

1. 完成预热
2. 采集基线参考
3. 进入 актив跃判定窗口
4. 观察 **专注** / **不专注** 输出随时间变化

有关状态机细节，请阅读 [参与度与专注度](/zh/docs/freebci-daq/engagement-focus)。

## 什么时候该升级到 8CH

如果你出现以下需求，就该进入 **8CH** 路径：

- 你需要的已不只是单通道 Demo
- 你需要同时覆盖多个点位
- 你想要更丰富的多通道监测与分析
