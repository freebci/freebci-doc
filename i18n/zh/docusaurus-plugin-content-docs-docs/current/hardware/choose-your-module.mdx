# 选择你的模块

> 先选最适合你目标的模块，再在同一个 DAQ 平台上逐步升级。

| 模块 | 最适合 | 你会得到什么 | 典型结果 |
|---|---|---|---|
| **FreeBCI Spike 1CH LH001** | 初学者、maker、课堂演示、快速迭代 | 单通道生物电采集 | 第一个信号、第一条波形、专注度 / 参与度 Demo |
| **FreeBCI Spike 8CH ADS1299** | EEG 搭建者、研究用户、多通道原型开发 | 八通道 EEG 采集 | 多通道波形、点位绑定、更丰富的分析 |

## 如果你满足这些条件，先从 1CH 开始

- 你希望最快得到一个可用的 FreeBCI 系统
- 你想优先演示 **参与度指数** 和二元 **专注度** 分类
- 你正在学习电极基础、信号质量和 DAQ 工作流
- 你想先走一条低复杂度路径，再升级到多通道 EEG

前往：[Spike 1CH LH001](/zh/docs/hardware/spike-1ch-lh001)

## 如果你满足这些条件，直接选择 8CH

- 你已经掌握基础，希望获得完整通道映射
- 你需要同时连接多个电极点位
- 你想进行更完整的空间监测与分析
- 你的目标是系统原型、多通道实验或完整演示

前往：[Spike 8CH ADS1299](/zh/docs/hardware/spike-8ch-ads1299)

## 共享软件平台

两个模块都由 **FreeBCI DAQ** 提供支持。

- 同样的浏览器采集流程
- 同样的 Web Serial 连接模型
- 同样的实时监测、EI、专注度和 AI 分析界面
- 不同的是硬件复杂度与搭建深度

通用软件入口见：[连接到 FreeBCI DAQ](/zh/docs/hardware/connect-to-freebci-daq)

## 推荐学习路径

1. 阅读 [安全与实验室环境](/zh/docs/hardware/safety-and-lab-setup)
2. 如果你是新手，先从 [Spike 1CH LH001](/zh/docs/hardware/spike-1ch-lh001) 开始
3. 跑通 [1CH 专注度 Demo](/zh/docs/hardware/spike-1ch-lh001/focus-demo)
4. 当你需要同时采集多个点位时，再进入 [Spike 8CH ADS1299](/zh/docs/hardware/spike-8ch-ads1299)
5. 准备升级时，阅读 [从 1CH 升级到 8CH](/zh/docs/hardware/upgrade-from-1ch-to-8ch)
