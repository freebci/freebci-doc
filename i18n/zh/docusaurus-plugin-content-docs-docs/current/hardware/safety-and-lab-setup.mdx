# 安全与实验室环境

> 在给板卡上电、连接电极或进行任何人体连接实验前，请先阅读本页。

## 适用范围

FreeBCI Spike 的公开资料和基于其派生的硬件，适用于：

- 科研
- 教学
- 工程原型开发

它们**不是**医疗器械资料包，也**不适用于**诊断、治疗、生命支持或其他高风险医疗用途。

## 人体连接规则

如果测试系统需要连接人体：

- 优先使用**电池供电**
- 采用保守的电气隔离方案
- 避免直接连接到市电供电设备
- 避免临时拼凑的接地方式
- 评估**整个链路**：板卡、电池、USB 接口、主机、充电器以及任何实验室仪器

## Bring-up 顺序

建议按阶段推进：

1. 上电前先检查板卡
2. 不接电极的情况下给板卡上电
3. 确认可见状态正常
4. 将板卡连接到 FreeBCI DAQ
5. 确认数据流稳定
6. 只有在上述全部成立后，再进入人体连接实验

不要把第一次验证直接做成“人体连接测试”。

## 实验台检查清单

第一次上电前，请确认：

- 模块版本和公开资料对应正确
- 焊点干净，连接器方向正确
- 没有可见短路、反装器件或电池接线错误
- 使用稳定电池或认可的隔离供电方式
- 实验硬件和市电实验设备保持清晰隔离

## USB 风险

USB 供电的开发环境如果处理不当，可能形成不安全的地回路或漏电路径。应将主机电脑、充电器、USB 线以及相关仪器视为同一个风险面的一部分。

## 责任归属

任何制造、装配、修改或使用派生硬件的个人或组织，都应自行负责：

- 安全
- 验证
- 合规
- 适用场景判断
