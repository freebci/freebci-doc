# FreeBCI Spike 8CH ADS1299

> FreeBCI 硬件中的多通道路径：面向更完整的 EEG 搭建、更丰富的点位映射，以及在 FreeBCI DAQ 中进行更完整的分析。

## 产品定位

FreeBCI Spike 8CH ADS1299 是 FreeBCI 产品线中的**进阶多通道模块**。

它最适合：

- 多通道 EEG 原型开发
- 通道与点位映射工作流
- 更丰富的实时监测与分析
- 已经完成 1CH Demo、准备升级的用户

## 公开发布内容

当前公开资料包包括：

- 原理图 PDF
- Interactive BOM HTML
- Gerber ZIP

当前**不包括**可编辑 EDA 源文件、固件源码、外壳文件、量产测试流程或监管文档。

## 硬件范围

这个模块基于 **ADS1299**，目标是八通道采集工作流。最终的电气解释、连接器定义和网络级细节，应始终以已发布原理图为准。

## 什么算成功

这个模块的成功首轮路径是：

1. 板卡上电正常
2. FreeBCI DAQ 建立稳定串口连接
3. 通道与点位绑定配置正确
4. 多条实时波形同时流动
5. 监测和分析体现出真正的多通道工作流
