# 8CH 首次上电

> 在尝试完整电极搭建前，先把硬件当作一个系统来验证。

## 上电前

请检查：

- 板卡与目标 8CH 发布版本一致
- 模拟前端焊接质量良好
- 连接器方向正确
- 没有可见短路或电池接线错误
- 测试环境符合隔离原型开发的安全要求

## 首次 bring-up 目标

第一次目标还不是“完整 EEG”，而是：

- 供电稳定
- 没有热故障或明显硬件异常
- 运行稳定到足以进入 DAQ 工作流

## 推荐 bring-up 顺序

1. 不接电极的情况下给板卡上电
2. 观察是否有明显故障表现
3. 将板卡接入 DAQ 环境
4. 确认设备可以通过 USB 串口被打开
5. 在开始严肃采集前，先准备完整通道绑定

## 为什么 8CH 需要更严格

相比 1CH，多通道板卡会带来更多 setup 质量敏感点：

- 每个通道都可能独立出问题
- 点位映射会直接影响解释
- 不同通道的噪声、参考质量和导联质量可能不同

因此，分阶段 bring-up 会在后面节省大量排查时间。
