# FreeBCI Spike

> 从单通道专注度 Demo 开始，再在同一个 DAQ 平台上升级到完整的多通道 EEG。

当前 FreeBCI 硬件主要围绕两条路径展开：

- **FreeBCI Spike 1CH LH001** — 面向入门用户、maker 和快速专注度 / 参与度 Demo 的起步路径
- **FreeBCI Spike 8CH ADS1299** — 面向完整多通道 EEG 采集与分析的进阶路径

两者都由 **FreeBCI DAQ** 提供支持。

## 从这里开始

| 路径 | 适合什么目标 | 前往 |
|---|---|---|
| 了解整体路线 | 在 1CH 与 8CH 之间做选择 | [选择你的模块](/zh/docs/hardware/choose-your-module) |
| 安全起步 | 了解电池、USB、实验室环境和人体连接注意事项 | [安全与实验室环境](/zh/docs/hardware/safety-and-lab-setup) |
| 共享软件流程 | 连接任一模块到浏览器中的 DAQ 工作流 | [连接到 FreeBCI DAQ](/zh/docs/hardware/connect-to-freebci-daq) |

## 产品逻辑

推荐的使用路径是：

1. 先从 **1CH** 开始，熟悉工作流并完成专注度 Demo
2. 需要同时采集多个点位时，再升级到 **8CH**
3. 在整个过程中保持同一套 **FreeBCI DAQ** 软件流程

继续阅读：[从 1CH 升级到 8CH](/zh/docs/hardware/upgrade-from-1ch-to-8ch)
