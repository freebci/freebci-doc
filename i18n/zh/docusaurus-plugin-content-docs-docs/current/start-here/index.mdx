# 开始这里

> 先选择最适合你的路径：单通道首个信号 Demo、完整八通道 EEG 搭建，或两者共享的 FreeBCI DAQ 工作流。

## 选择你的路径

### 1CH：第一个信号与专注度 Demo

适合：

- 初学者
- maker
- 课堂演示
- 希望快速上手的用户

目标：

- 安全上电
- 接入一个有效通道
- 看到实时波形
- 跑通专注度 / 参与度 Demo

建议从这里开始：

1. [选择你的模块](/zh/docs/hardware/choose-your-module)
2. [1CH 快速路径](#1ch-快速路径)
3. [安全与实验室环境](/zh/docs/hardware/safety-and-lab-setup)
4. [Spike 1CH LH001](/zh/docs/hardware/spike-1ch-lh001)
5. [连接到 FreeBCI DAQ](/zh/docs/hardware/connect-to-freebci-daq)
6. [1CH 专注度 Demo](/zh/docs/hardware/spike-1ch-lh001/focus-demo)

### 8CH：完整多通道 EEG

适合：

- 进阶用户
- EEG 系统搭建者
- 多通道原型开发
- 需要更完整点位监测的用户

目标：

- 安全搭建 8CH 模块
- 正确完成多点位绑定
- 验证多通道稳定采集
- 使用 FreeBCI DAQ 进行监测与调参

建议从这里开始：

1. [选择你的模块](/zh/docs/hardware/choose-your-module)
2. [8CH 快速路径](#8ch-快速路径)
3. [安全与实验室环境](/zh/docs/hardware/safety-and-lab-setup)
4. [Spike 8CH ADS1299](/zh/docs/hardware/spike-8ch-ads1299)
5. [连接到 FreeBCI DAQ](/zh/docs/hardware/connect-to-freebci-daq)
6. [8CH 多通道 EEG 搭建](/zh/docs/hardware/spike-8ch-ads1299/multichannel-eeg-setup)

### 同一个 DAQ，不同硬件复杂度

两条硬件路径都使用同一套软件平台：

- [FreeBCI DAQ 概览](/zh/docs/freebci-daq)
- [硬件配置](/zh/docs/freebci-daq/hardware-setup)
- [实时监测](/zh/docs/freebci-daq/live-monitoring)
- [参与度与专注度](/zh/docs/freebci-daq/engagement-focus)

## 1CH 快速路径

如果你的目标是尽快跑通一个可用 Demo：

1. 阅读 [安全与实验室环境](/zh/docs/hardware/safety-and-lab-setup)
2. 打开 [Spike 1CH LH001](/zh/docs/hardware/spike-1ch-lh001)
3. 按 [连接到 FreeBCI DAQ](/zh/docs/hardware/connect-to-freebci-daq) 进入软件流程
4. 运行 [1CH 专注度 Demo](/zh/docs/hardware/spike-1ch-lh001/focus-demo)

## 8CH 快速路径

如果你的目标是完整多通道 EEG 工作流：

1. 阅读 [安全与实验室环境](/zh/docs/hardware/safety-and-lab-setup)
2. 打开 [Spike 8CH ADS1299](/zh/docs/hardware/spike-8ch-ads1299)
3. 按 [连接到 FreeBCI DAQ](/zh/docs/hardware/connect-to-freebci-daq) 进入软件流程
4. 完成 [8CH 多通道 EEG 搭建](/zh/docs/hardware/spike-8ch-ads1299/multichannel-eeg-setup)
