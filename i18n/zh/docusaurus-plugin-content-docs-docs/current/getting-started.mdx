# 快速开始

如果你是第一次接触 FreeBCI，请先选择最适合你的路径。

## 路径 1：1CH 入门

如果你希望最快跑通一个可工作的 Demo，请选择这条路径。

- 适合初学者、maker 和课堂演示
- 目标：安全上电、看到波形、跑通专注度 / 参与度 Demo

从这里开始：

1. [开始这里](/zh/docs/start-here)
2. [Spike 1CH LH001](/zh/docs/hardware/spike-1ch-lh001)
3. [1CH 专注度 Demo](/zh/docs/hardware/spike-1ch-lh001/focus-demo)

## 路径 2：8CH 多通道 EEG

如果你希望搭建一个更完整的多通道 EEG 系统，请选择这条路径。

- 适合进阶用户、EEG 搭建者和多通道原型开发
- 目标：完成多点位绑定、验证多通道采集、获得更丰富的监测视图

从这里开始：

1. [开始这里](/zh/docs/start-here)
2. [Spike 8CH ADS1299](/zh/docs/hardware/spike-8ch-ads1299)
3. [8CH 多通道 EEG 搭建](/zh/docs/hardware/spike-8ch-ads1299/multichannel-eeg-setup)
