## About

FreeBCI is an open-source brain-computer interface platform. Both the hardware designs and software are fully open source, built by a global community of researchers, engineers, and makers.

This repository contains the official website and documentation portal, built with [Docusaurus 3](https://docusaurus.io/) and [Tailwind CSS](https://tailwindcss.com/).

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) >= 18

### Installation

```sh
git clone https://github.com/freebci/freebci-doc.git
cd freebci-doc
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
